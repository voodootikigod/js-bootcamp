JS Boot Camp - Empire Robots Edition

From Interested To Capable Hardware Hacking In Four Hours

Presented By Chris Williams - Curator of JSConf and Creator of Node-Serialport

Lab Assistant: Rick Waldron - Creator of Johnny-five

Pivotal Labs in New York City, NY - October 21st

JS is an incredibly powerful and widely used scripting language that powers the web, but did you know that it can also be used to power robots? This is a targeted, rapid learning session for exposing software developers, specifically JS developers, to the wonderfully wide world of hardware hacking. There is little more satisfying than creating real-world objects and putting them in motion with familiar programming code structures. No previous hardware development knowledge is required and only mild JS experience is necessary. We are offering this course in conjunction with EmpireJS so be sure to register for that as well! If you have ever been curious about all the energy and increasing excitement around hardware hacking, this is the perfect event for you to get started hacking!

The class will require that you bring a computing device that has node.js version 0.8.12 pre-installed along with NPM. That is the only pre-requisite for attendees. We are offering two price points, the first price point ($165.00) includes the Sparkfun Inventor's Kit which you will be able to take home with you. The second price point ($65.00) does not include the Sparkfun Inventor's Kit and requires you to bring this kit with you. Please note: if you show up without the proper kit and opted for the lower price, you will be able to sit in and learn, but you won't be able to enjoy the full hands-on experience of this class. If you do not have the identified kit, please be sure to sign up for the first price point.

As part of this course you will become comfortable with hardware programming using node.js including an survey of the following topics:

  • Serial Port Interface (SPI)
  • Breadboarding
  • Reading Resistors
  • Arduino
  • The Firmata Protocol
  • Inter-Integrated Circuits (I2C)
  • General Purpose I/O (GPIO)
  • Connecting devices to the web
  • Controlling Drones
